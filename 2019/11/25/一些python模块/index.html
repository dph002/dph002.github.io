如何查找python库的老版本

在scikit-image中，网上的例子有一个标准的美女图，lena，但是在scikit-image 0.10.0之后的版本中没有了，这样，我们到https://github.com/scikit-image/scikit-image，中点击branches，分支，然后查找带0.10.x的版本，下载下来，里面就有该lena.png的文件，导出使用即可
