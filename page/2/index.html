hdf5删除数据集的问题

HDF5中没有提供直接删除数据集的功能，可以通过以下方式进行删除

  1. 程序以写方式打开infile.h5，打开指定的Group,调用删除到数据集的链接函数(h5ldelete),保存回原文件
  2. 在命令行执行h5repack infile.hg outfile.h5，outfile.h5中不会保存失去连接的数据集，这样变通的方式

Jupyter安装多Kernel

jupyter 6.0 以后的版本

  • 可以安装 python3.5以上的kernel

  • Julia内核

  • R 内核

Jupyter 5.0 一下的版本：

  • python 2.7 内核
  • Julia 内核
  • R 内核

用conda的activate py36 activate python27等等，可以维护多个版本的python

在用py36或者py27的版本打开jupyter的notebook时，Julia 和R内核都可以，直接用，因为该信息是写在用户空间的共同配置文件中的。
